# word-tester 单词背诵系统

本工具是由 Justskr 开发，用于背诵单词，支持自定义单词文件的单词背诵系统。完全开源免费，您可以在 GitHub 上查看项目代码。

## 注意事项

本项目使用纯前端技术实现，不依赖任何后端服务器，因此不会留存用户的任何数据，刷新页面后数据会丢失，请谨慎操作。

## JSON文件格式要求

本系统仅支持JSON格式的单词文件，请按照以下格式准备您的单词文件。

### 标准格式

```json
[
  {
    "en": "apple",
    "cn": "苹果"
  },
  {
    "en": "banana",
    "cn": "香蕉"
  },
  {
    "en": "orange",
    "cn": "橙子"
  }
]
```

### 字段说明

- `en`: 英文单词（必填）
- `cn`: 中文释义（必填）

### 注意事项

1. JSON文件必须是有效的JSON格式
2. 必须包含 `en` 和 `cn` 两个字段
3. 建议使用UTF-8编码保存文件
4. 文件扩展名必须为 `.json`

### 示例文件

项目目录中提供了两个示例JSON文件：
- `test-words.json` - 水果类单词示例
- `programming-words.json` - 编程相关单词示例

### 创建自己的单词文件

您可以使用任何文本编辑器创建JSON文件，也可以使用在线JSON编辑器。

推荐格式：
```json
[
  {
    "en": "your_english_word",
    "cn": "您的中文释义"
  }
]
```

### 常见错误

1. **缺少引号**：确保所有键和值都使用双引号
2. **逗号错误**：最后一个元素后不要有逗号
3. **字段名称错误**：必须使用 `en` 和 `cn`，不能使用其他名称
4. **编码问题**：确保文件使用UTF-8编码
