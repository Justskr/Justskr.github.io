# wordskr 单词背诵系统

![wordskr Logo](img/Logo.png)

## 项目简介

wordskr 是一款由 Justskr 开发的单词背诵系统，支持用户通过自定义单词文件进行高效学习。该项目完全开源免费，您可以在 GitHub 上查看完整的项目代码。

### 技术特点

- **纯前端实现**：不依赖任何后端服务器
- **数据隐私**：所有数据均存储在浏览器本地，不会被上传至任何服务器
- **自定义词库**：支持用户上传符合特定格式的 JSON 单词文件
- **简单易用**：直观的用户界面，便于快速上手

## 重要说明

由于本项目采用纯前端技术实现，所有数据仅保存在浏览器内存中。**刷新页面后所有学习数据将会丢失**，因此在使用过程中请谨慎操作，建议定期导出或备份您的学习进度。

## JSON 单词文件格式规范

本系统仅支持 JSON 格式的单词文件，请严格按照以下格式准备您的单词库。

### 标准格式

```json
[
  {
    "en": ["apple"],
    "cn": {
      "n. 苹果": []
    },
    "like": [],
    "note": ""
  },
  {
    "en": ["banana"],
    "cn": {
      "n. 香蕉": []
    },
    "like": [],
    "note": ""
  },
  {
    "en": ["orange"],
    "cn": {
      "n. 橙子": []
    },
    "like": [],
    "note": ""
  }
]
```

### 字段说明

| 字段名 | 类型           | 描述                               | 是否必填 |
|--------|----------------|------------------------------------|----------|
| `en`   | string 或 array | 英文单词，支持单个字符串或字符串数组（多个拼写） | 是       |
| `cn`   | object         | 中文释义对象，键为包含词性的释义，值为示例数组 | 是       |
| `like` | array          | 相似单词或相关词汇（可选）           | 否       |
| `note` | string         | 备注信息（可选）                     | 否       |

### 格式要求

1. **JSON 有效性**：必须是标准的 JSON 格式，确保所有键值对使用双引号包围
2. **字段完整性**：每个单词对象必须包含 `en` 和 `cn` 两个字段
3. **编码规范**：建议使用 UTF-8 编码保存文件，确保中文字符正常显示
4. **文件扩展名**：必须使用 `.json` 作为文件扩展名
5. **数组结构**：单词列表必须包含在方括号 `[]` 中，每个单词对象之间使用逗号分隔
6. **词性格式**：中文释义的键建议使用 "词性. 释义" 格式，如 "n. 苹果"、"v. 跑"

### 创建自定义单词文件

您可以使用任何文本编辑器创建 JSON 文件，也可以使用在线 JSON 编辑器来确保格式正确。

#### 推荐创建步骤

1. 打开文本编辑器（如 Notepad++、VS Code 等）
2. 输入标准 JSON 格式的单词数据
3. 使用 UTF-8 编码保存文件
4. 将文件扩展名设置为 `.json`
5. 在系统中上传并使用该文件

#### 最小示例

```json
[
  {
    "en": ["word"],
    "cn": {
      "n. 单词": []
    }
  }
]
```

#### 高级示例（支持多个拼写和多个释义）

```json
[
  {
    "en": ["color", "colour"],
    "cn": {
      "n. 颜色": [],
      "v. 给...着色": []
    },
    "like": ["hue", "shade"],
    "note": "美国英语拼写为color，英国英语拼写为colour"
  }
]
```

### 常见错误及解决方案

| 错误类型         | 表现形式                          | 解决方案                              |
|------------------|-----------------------------------|---------------------------------------|
| 缺少引号         | JSON 解析错误                     | 确保所有键和字符串值都使用双引号包围  |
| 多余逗号         | 末尾元素后有逗号                  | 移除最后一个元素后的逗号              |
| 字段名称错误     | 字段名不是 `en` 或 `cn`           | 修正字段名称为标准格式                |
| 编码问题         | 中文字符显示异常                  | 使用 UTF-8 编码保存文件               |
| 格式不规范       | 方括号或花括号不匹配              | 检查并修正 JSON 结构                  |
| 词性格式不正确   | 中文释义显示异常                  | 确保 `cn` 对象的键使用 "词性. 释义" 格式 |

## 使用指南

1. **准备单词文件**：按照上述格式创建您的 JSON 单词文件
2. **上传文件**：在系统中选择并上传您的单词文件
3. **开始学习**：系统会自动加载单词并开始学习流程
4. **查看进度**：学习过程中可以随时查看当前的学习状态和统计信息

## 最佳实践

- **合理分组**：建议将单词按照主题或难度进行分组，创建多个单词文件
- **适量添加**：每次学习的单词数量不宜过多，建议控制在 20-50 个单词之间
- **定期复习**：结合系统的统计功能，定期复习出错率较高的单词
- **备份文件**：妥善保管您的单词文件，避免意外丢失
- **详细释义**：为单词添加完整的词性和释义，有助于更全面地理解单词
